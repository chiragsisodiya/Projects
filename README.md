# Credit Scoring System

Colab notebook project

## TL;DR

This repository contains a Google Colab notebook that implements a credit scoring system (data processing, modeling, and evaluation). The notebook is ready to run in Colab or locally.

## Files included

- `CREDIT_SCORING_SYSTEM.ipynb` — main Colab notebook (this project).
- `requirements.txt` — inferred Python packages used in the notebook.
- `LICENSE` — MIT license (template).
- `COMMIT_MESSAGE.txt` — suggested commit message for GitHub.

## Run in Google Colab (recommended)

1. Open https://colab.research.google.com/
2. Click **File → Upload notebook** and select `CREDIT_SCORING_SYSTEM.ipynb`, OR use **GitHub** tab after you push this repo.

## Run locally

1. Create a virtual environment and install dependencies:
```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

2. Open the notebook in JupyterLab / Jupyter Notebook and run cells, or convert to script if needed.
